## Installation.

### Clone this repository

```
git clone git@github.com:arberLisaj/search-users.git
```

```
cd search-users
```

### Start the server

```
cd server
```

```
npm i
```

```
node .
```

### Start the client / react-app

- Open a new terminal window inside the search-users project to start the dev server and run:

```
npm i
```

```
npm run dev
```
