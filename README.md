## Installation.
### Clone this repository

```
git clone git@github.com:arberLisaj/search-users.git
```
```
cd search-users
```
### Start the server
```
cd server
```
```
npm i
```
```
node .
```
### Start the client / react-app
- Open a new terminal window and cd inside the project.
```
cd search-users
``` 
```
npm i
```
```
npm run dev
```
